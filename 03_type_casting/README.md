## 3. 转型演示

```py
name = "波波"
age = 28
gpa = 3.9
is_student = True
```

### 3.1 在Python中，如何显示一个变量的类型？[ask_copilot]

```py
print(type(name))
print(type(age))
print(type(gpa))
print(type(is_student))
```
