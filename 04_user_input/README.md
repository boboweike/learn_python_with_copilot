## 1. 问题：在 Python 中，如何获取用户输入？[Ask_copilot]

## 2. 用户输入演示

```py
# 请输入你的名字name
# 请输入你的年龄age

# 打印输出你好{name}，你的年龄是{age}岁
```

需要转型：

```py
name = input('请输入名字：')
age = input('请输入年龄：')
age = age + 1
print(f'你好{name}，你的年龄是{age}岁')
```
