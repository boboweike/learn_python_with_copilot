# 信用卡号校验程序

```py
# 1. 请用户输入一个信用卡号credit_card_number
# 2. 移除信用卡号中的非数字字符，比方说空格和连字符，留下纯数字。
# 3. 将信用卡号从最后一个数字开始反转顺序。
# 4. 将奇数位数字求和，结果为奇数位控制数 oddSum。
# 5. 将偶数位数字按下面的规则进行转换：
#     - 将每个数字都乘以 2。
#     - 如果乘以 2 后的数字比 9 大，就减去 9。
#     - 将偶数位数字求和，结果为偶数位控制数 evenSum。
# 6. sum = oddSum + evenSum。
# 7. 如果 sum 末尾是 0，信用卡号是合法的；否则不合法。
```
