# 小卖部程序

```py
# 1. 定义和初始化变量
# 1.1 定义菜单menu字典变量，以键值对的形式存储商品和价格信息
#      商品和价格分别是：'矿泉水':2.0, '可乐':3.5, '雪碧':3.0, '面包':4.9, '饼干':5.0, '巧克力':6.5
# 1.2 定义购物车cart列表，用于存储购买的商品名称
# 1.3 定义总金额total_price变量，用于存储购买商品的总金额

# 2. 打印菜单
# 2.0 打印菜单标题：----商品清单----
# 2.1 遍历菜单menu字典，打印商品和价格信息，格式为：商品名称：¥价格，
#     商品名称占10个字符位，价格保留2位小数

# 3. 用户购物
# 3.1 打印分隔线：----------------
# 3.2 提示用户输入商品名称，如果输入为q/Q，结束循环
# 3.3 如果输入的商品名称在菜单menu字典中，将商品名称添加到购物车cart列表中
# 3.4 如果输入的商品名称不在菜单menu字典中，提示用户输入错误，重新输入

# 4. 结算
# 4.1 打印提示信息：----你的订单----
# 4.2 遍历购物车cart列表，打印购买的商品名称，商品之间用逗号分隔
# 4.3 遍历购物车cart列表，计算购买商品的总金额total_price
# 4.4 打印："你购买的总金额是: {total_price}元"，保留2位小数
```
